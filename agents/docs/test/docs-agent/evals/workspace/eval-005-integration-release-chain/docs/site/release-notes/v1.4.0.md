# v1.4.0

## 用户可见功能

AI Hub 新增只读知识搜索接口，支持非空查询词与最多 20 条结果。

## 架构与接口

新增 `GET /api/search`，非法空查询返回结构化 400 错误。

## 数据库与交付资产

本版本没有数据库迁移、镜像或静态资产变化。

## 部署、升级与兼容性

无需新增环境变量或升级步骤；接口为新增能力，不改变既有 API。

## 风险与回滚

当前返回为空结果集合的最小实现；回滚时移除搜索路由，不涉及数据恢复。

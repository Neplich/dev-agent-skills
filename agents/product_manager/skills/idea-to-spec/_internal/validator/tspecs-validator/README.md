# TEST_SPEC Validator (tspecs-validator)

## Overview

Validate test specifications for requirement coverage, traceability, and
test-case completeness.

## When to use

- Before QA handoff
- After updating test cases
- When you need a coverage and traceability review

## Quick start

```text
Validate this TEST_SPEC and tell me where coverage is missing.
```

## Related skills

- `tspecs-gen`
- `tspecs-iteration`
- `trace-check`

# TRD Iteration (trd-iteration)

## Overview

Analyze a TRD revision request and hand it to `engineer-agent:trd-gen`. PM does
not update Engineer-owned TRD content directly.

## When to use

- `trd-validator` found issues
- Architecture review requires changes
- Data model, API, NFR, deployment, or security design changed

## Quick start

```text
Analyze this TRD validator report and prepare the Engineer handoff packet.
```

## Related skills

- `engineer-agent:trd-gen`
- `trd-validator`
- `adr-iteration`
- `api-iteration`

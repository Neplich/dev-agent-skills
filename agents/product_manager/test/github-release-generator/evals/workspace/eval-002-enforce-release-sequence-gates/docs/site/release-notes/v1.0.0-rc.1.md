# AI Hub v1.0.0-rc.1

本候选版本新增文件卡片和失败消息原位重试。Gateway 将 `workflow_finished` 转换为统一附件模型，同时保留旧文本消息兼容路径。

数据库新增 nullable JSONB `message_files`，先加列后回填；删除该列会丢失附件元数据。部署顺序为数据库、Gateway、Web，文件卡片开关生产默认关闭。

Web 与 Gateway 镜像均提供 amd64/arm64，静态 manifest 包含 file-card chunk。升级应先备份并分步冒烟；旧浏览器不显示流式下载进度，但普通下载可用。

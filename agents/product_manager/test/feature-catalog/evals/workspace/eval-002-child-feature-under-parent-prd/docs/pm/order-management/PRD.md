# Order Management PRD

Order management covers order placement, order detail, and order status
tracking for the retail platform.

## Scope

- Place an order from the storefront
- Query order detail and status
- Track status transitions: created -> paid -> shipped -> completed

Refund handling is out of scope for this version and will be specified as a
follow-up.

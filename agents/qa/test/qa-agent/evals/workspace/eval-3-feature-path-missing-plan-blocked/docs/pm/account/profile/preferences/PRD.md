# Profile Preferences PRD

Users can update communication preferences from the profile settings area. This
is a third-level feature under Account > Profile and must not be treated as a
new top-level QA scope.

Acceptance requires documented E2E coverage for saving preferences, canceling
changes, and displaying validation feedback.

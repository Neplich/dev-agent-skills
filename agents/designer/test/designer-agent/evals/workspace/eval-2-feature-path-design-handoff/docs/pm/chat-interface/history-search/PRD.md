# Chat History Search PRD

Users need to search within existing chat history by keyword, participant, and
date range from the Chat Interface. The work is a child feature of
`chat-interface`, not a new top-level product area.

Design must preserve the existing chat layout while adding a search entry point,
results list, empty state, and clear return path to the conversation.

# Product Patterns

Choose the nearest product category and adapt it to the user's context.

| Product category | Recommended pattern | Visual priority | Notes |
| --- | --- | --- | --- |
| SaaS General | Hero + product preview + feature proof | Trust, clarity, conversion | Use restrained brand color and direct CTA hierarchy. |
| Enterprise Analytics Dashboard | Data-Dense Dashboard | Scanability, comparison, status clarity | Prioritize tables, filters, metrics, trend visibility, and low-noise surfaces. |
| Financial / Fintech | Trust + Status Clarity | Confidence, risk signaling, legibility | Avoid playful colors and ambiguous red/green usage. |
| Healthcare | Calm Guided Workflow | Accessibility, reassurance, error prevention | Use human tone, clear recovery paths, and conservative status colors. |
| E-commerce | Feature-Rich Showcase | Product discovery, conversion, trust | Emphasize imagery, filters, cart feedback, and urgency without clutter. |
| Consumer Social / Community | Content-First Feed | Personality, creation, engagement | Keep creation actions visible and reduce navigation friction. |
| Developer Tool | Dense Utility Workspace | Speed, precision, hierarchy | Prefer technical typography, shortcuts, command surfaces, and compact density. |
| Education / Learning | Guided Progression | Motivation, clarity, feedback | Use progress markers, friendly feedback, and digestible modules. |

# 管理面板

## 当前状态

管理面板通过 `/admin` 下的服务端 API 提供用户列表和账号管理能力。管理员登录后才能访问这些端点。

## 已确认范围

- 审查管理路由注册、认证中间件和管理员角色校验。
- 确认匿名用户与普通用户无法进入管理处理器。
- 安全报告需要包含代码位置、影响、严重度依据与可执行修复建议。

## 非目标

- 修改管理面板视觉设计。
- 增加新的账号管理操作。

## 验收期望

- 所有 `/admin` 端点必须先验证会话身份，再验证 `admin` 角色。
- 未认证请求和非管理员请求必须在执行管理查询前被拒绝。

# 用户数据权利

## 当前状态

产品提供资料读取、数据导出和账号删除入口，但尚未确认这些入口是否覆盖全部个人数据及其第三方副本。

## 目标

- 已登录用户只能访问和导出自己的数据。
- 删除请求覆盖主库、后台任务和第三方分析副本。
- 导出包含用户可识别的资料、订单和行为数据，并使用安全交付方式。

## 范围

- 用户资料访问、导出与删除端点。
- 主数据库和分析系统中的个人数据副本。
- 不改变现有认证模型。

## 验收期望

- 所有端点从可信会话获取用户身份。
- 删除状态可追踪，并明确依法必须保留的数据范围与期限。
- 导出完整、机器可读且不会暴露其他用户数据。

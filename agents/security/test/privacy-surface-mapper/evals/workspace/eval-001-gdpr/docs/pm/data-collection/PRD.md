# 个人数据收集

## 当前状态

注册流程收集姓名和电子邮箱，服务日志与产品分析还会处理 IP 地址、设备信息和行为事件。当前尚未形成完整的数据处理清单。

## 目标

- 明确个人数据类型、收集入口和处理目的。
- 核对同意、数据最小化、第三方共享与保留规则。
- 检查访问、删除和导出等用户权利是否覆盖所收集数据。

## 范围

- 注册处理器、应用日志和分析配置。
- 账号创建与产品使用分析目的。
- 只输出隐私处理面与整改建议，不提供法律意见。

## 验收期望

- 数据清单能追溯到具体代码或配置。
- 未声明目的、同意、保留或共享的处理被标为缺口。
- 建议覆盖数据最小化和用户权利实现。
